Quando Usar

Esse é o cenário mais comum do Mirror: a origem já existe e você precisa gerar um novo objeto de destino preenchido com base nela.

Exemplo

var mirror = new Mirror.Mirror();

var user = new User
{
    Id = 1,
    Name = "Ana",
    Email = "ana@company.com"
};

var dto = mirror.Reflect<User, UserDto>(user);

O que Acontece no Pipeline

  • o destino é criado pelo Mirror
  • as propriedades compatíveis são localizadas por nome
  • valores simples são copiados
  • objetos complexos e coleções entram no pipeline recursivo
