Aplicar Transformações Customizadas

Voltar para Formas de Utilização

Objetivo

Transformações permitem que uma propriedade do destino seja montada a partir de uma regra explícita sobre o objeto de origem. Esse recurso é indicado quando o valor não deve ser uma cópia literal.

Exemplo

var configuration = new MirrorConfiguration();

configuration.CreateReflection<User, UserDto>()
    .ForMember(dto => dto.Name, user => user.Name.ToUpperInvariant());

var mirror = new Mirror.Mirror(configuration);
var dto = mirror.Reflect<User, UserDto>(user);

Cenários Típicos

  • normalização de texto
  • junção de campos
  • formatação de propriedades para exibição
  • adaptação simples entre contratos diferentes
