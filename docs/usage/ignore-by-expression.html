Ignorar Propriedades por Expressão

Voltar para Formas de Utilização

Objetivo

Ignorar por expressão é útil quando a regra de preservação depende da chamada atual e não deve ficar fixa no tipo de destino. A lista de membros ignorados é passada diretamente na invocação do Reflect.

Exemplo Atualizando Objeto Existente

mirror.Reflect(source, destination, x => x.Name);
mirror.Reflect(source, destination, x => x.Name, x => x.Windows);

Exemplo Criando Novo Objeto

var dto = mirror.Reflect<House, HouseDto>(source, x => x.Name);

Quando Preferir esse Formato

  • quando a regra é contextual
  • quando o mesmo tipo de destino pode ser usado em fluxos diferentes
  • quando o projeto não quer acoplar a regra ao modelo com atributo
