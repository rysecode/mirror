Configurar um Mirror Padrão para as Extensions

Voltar para Formas de Utilização

Objetivo

As extensions sem parâmetro explícito de IMirror dependem de um mirror padrão configurado. Isso permite chamadas como source.Reflect<Destination>() sem precisar injetar o mapper em toda parte.

Exemplo

var configuration = new MirrorConfiguration();

configuration.CreateReflection<User, UserDto>()
    .ForMember(dto => dto.Name, user => user.Name.ToUpperInvariant());

MirrorExtensions.SetDefaultMirror(new Mirror.Mirror(configuration));

var dto = user.Reflect<UserDto>();

MirrorExtensions.ResetDefaultMirror();

Cuidados

  • como o mirror padrão é global, use essa estratégia com disciplina
  • em aplicações maiores, prefira definir claramente onde a API fluida é permitida
  • em cenários altamente concorrentes, avalie se a configuração global atende bem ao modelo da aplicação
