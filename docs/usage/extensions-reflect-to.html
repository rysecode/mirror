Usar Extensions para Atualizar Destino

Voltar para Formas de Utilização

Exemplo Básico

using Mirror.Extensions;

user.ReflectTo(existingDto);

Exemplo com Ignore por Expressão

user.ReflectTo(existingDto, x => x.Name);

Observação

Esse formato é especialmente útil quando o destino já existe e a aplicação deseja manter a fluidez da sintaxe de extensão.
