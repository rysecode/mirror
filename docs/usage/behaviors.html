IgnoreNullValues

Quando habilitado, valores nulos vindos da origem não sobrescrevem o que já existe no destino.

var config = new MirrorConfiguration
{
    IgnoreNullValues = true
};

MaxDepth

Define a profundidade máxima de mapeamento em grafos complexos.

var config = new MirrorConfiguration
{
    MaxDepth = 5
};
