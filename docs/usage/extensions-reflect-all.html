Mapear Coleções com Extensions

Voltar para Formas de Utilização

Objetivo

ReflectAll facilita o mapeamento de sequências inteiras sem exigir loop manual explícito no código de aplicação.

Exemplo

var users = new List<User>
{
    new() { Id = 1, Name = "Ana" },
    new() { Id = 2, Name = "Bruno" }
};

var dtos = users.ReflectAll<User, UserDto>().ToList();

Benefícios

  • reduz código repetitivo em loops de projeção
  • mantém o mesmo pipeline de mapeamento por item
  • funciona bem com coleções simples ou profundas
