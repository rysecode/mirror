Quando Usar

Use esse formato quando o objeto de destino já foi instanciado e não deve ser recriado. Isso é comum em operações de update, sincronização parcial ou reaproveitamento de instância já associada a algum contexto da aplicação.

Exemplo

var source = new User
{
    Id = 1,
    Name = "Ana",
    Email = "ana@company.com"
};

var destination = new UserDto
{
    Id = 999,
    Name = "Valor original",
    Email = "original@company.com"
};

mirror.Reflect(source, destination);

Benefícios

  • preserva a identidade do objeto de destino
  • permite combinar com ignores e IgnoreNullValues
  • funciona bem em cenários com objetos aninhados já existentes
