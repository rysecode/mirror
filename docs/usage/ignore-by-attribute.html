Ignorar Propriedades por Atributo

Voltar para Formas de Utilização

Objetivo

O atributo MirrorNonReflect permite marcar uma propriedade do destino que nunca deve ser alterada pelo pipeline de mapeamento. Esse formato é ideal quando a regra de preservação é estrutural e deve valer sempre para aquele membro.

Exemplo

public class HouseDto
{
    [MirrorNonReflect]
    public string Name { get; set; } = string.Empty;

    public int Doors { get; set; }
    public int Windows { get; set; }
}

Comportamento

Se o destino já possuir um valor em Name, esse valor será preservado mesmo que a origem tenha outro valor disponível para a mesma propriedade.
