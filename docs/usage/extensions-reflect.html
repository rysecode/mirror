Usar Extensions para Criar Novo Destino

Voltar para Formas de Utilização

Objetivo

As extensões oferecem uma sintaxe mais fluida e conveniente para o consumo da biblioteca. Com um mirror padrão configurado, é possível mapear diretamente a partir do objeto de origem.

Exemplo

using Mirror.Extensions;

var dto = user.Reflect<UserDto>();

Quando Esse Formato é Útil

  • quando a equipe prefere sintaxe fluida
  • quando existe um mirror padrão global definido
  • quando o código de aplicação precisa ficar mais enxuto
