Usar Factories

Voltar para Formas de Utilização

Quando Usar

Factories devem ser usadas quando a construção do destino precisa passar por uma regra própria, método estático, validação interna ou qualquer outra forma de criação que não seja simplesmente uma instanciação padrão.

Factory Registrada em Configuração

var configuration = new MirrorConfiguration();

configuration.AddFactory<CreateCompanyRequest, Company>(request =>
    Company.Create(request.Name, request.Document)
);

var mirror = new Mirror.Mirror(configuration);
var company = mirror.Reflect<CreateCompanyRequest, Company>(request);

Factory Informada Diretamente na Chamada

var result = mirror.ReflectWithFactory(request, source =>
    Company.Create(source.Name, source.Document)
);

Boas Práticas

  • use factory quando o destino possui invariantes
  • prefira configuração global se a regra for recorrente
  • prefira factory inline quando a regra for local ao fluxo
