MirrorException

try
{
    var dto = source.ReflectSafe<Destination>();
}
catch (MirrorException ex)
{
    Console.WriteLine(ex.Message);
    Console.WriteLine(ex.InnerException?.Message);
}

Falhas de mapeamento são encapsuladas em MirrorException, preservando a exceção original.

Proteção contra Recursão Excessiva

Grafos complexos podem conter profundidade elevada ou ciclos. Por isso, a implementação respeita MaxDepth e mantém controle do contexto durante a recursão.

Cuidados Práticos

  • configure MaxDepth de forma intencional
  • use IgnoreNullValues em updates parciais
  • para construções de domínio complexas, prefira factories
  • escreva testes específicos para cenários ambíguos do projeto
