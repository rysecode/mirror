Visão Geral

O Mirror é uma biblioteca de mapeamento objeto-objeto para .NET projetada para reduzir código repetitivo sem esconder o comportamento por trás de convenções excessivamente mágicas. A proposta é oferecer uma API pequena, previsível e capaz de atender desde cenários simples até grafos complexos com coleções, dicionários, profiles, factories e regras de ignore.

Esta área de documentação foi organizada para funcionar como uma referência técnica. Cada tema principal possui sua própria página, e cada forma de utilização relevante da biblioteca foi isolada em uma página específica para facilitar consulta, onboarding e manutenção.
